## 🎯 ¿Qué hace esta extensión?

Esta extensión **automatiza el proceso de búsqueda de citas previas** en el sistema oficial de Extranjería (`icp.administracionelectronica.gob.es`), ahorrándote horas de espera y reintentos manuales.

### 🔄 Flujo de trabajo automático:

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   INICIO    │────▶│  Provincia  │────▶│  Comisaría  │────▶│Datos Personales│
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
                                                                   │
                                                                   ▼
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   🔔 ALERTA │◀────│  ✅ CITA    │◀────│   🔍 Buscar  │◀────│  Contacto   │
│    SONORA   │     │ DISPONIBLE  │     │   CITAS     │     │   Email/Tel │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
                           │
                           ▼
                    ┌─────────────┐
                    │ 👤 USUARIO  │
                    │  COMPLETA   │
                    │   CAPTCHA   │
                    └─────────────┘
```

## 🚀 Instalación

### 📦 Requisitos Previos

- Navegador compatible (Chrome, Edge, Brave, Opera)
- Conexión a internet
- Cuenta de correo electrónico

### 🟢 Google Chrome

**Paso 1:** Descarga el código desde GitHub o el ZIP

**Paso 2:** Abre Chrome y ve a `chrome://extensions/`

**Paso 3:** Activa "Modo desarrollador" (esquina superior derecha)

**Paso 4:** Haz clic en "Cargar descomprimida" y selecciona la carpeta

**Paso 5:** ¡Listo! La extensión aparecerá en tu barra de herramientas

### 🔵 Microsoft Edge

**Paso 1:** Ve a `edge://extensions/`

**Paso 2:** Activa "Modo desarrollador"

**Paso 3:** Clic en "Cargar descomprimida" y selecciona la carpeta

### 🟣 Brave / Opera

**Brave:** `brave://extensions/`

**Opera:** `opera://extensions/`

## ⚙️ Configuración

### 1️⃣ Completa tus Datos Personales

```
┌─────────────────────────────────────────────────────────────┐
│  📄 N.I.E.        │  X1234567L                              │
│  👤 NOMBRE        │  JUAN PEREZ GARCIA                       │
│  🌍 NACIONALIDAD  │  ESPAÑA (348)                           │
│  📍 PROVINCIA     │  Madrid                                 │
│  ✉️ EMAIL         │  juan.perez@ejemplo.com                 │
│  📞 TELÉFONO      │  612345678                              │
└─────────────────────────────────────────────────────────────┘
```

### 2️⃣ Selecciona tus Comisarías

1. Haz clic en **"🏢 COMISARÍAS"** para expandir
2. Marca las oficinas que te interesan
3. Puedes seleccionar múltiples comisarías

### 3️⃣ Configura Horarios (Opcional)

| Modo | Descripción |
|------|-------------|
| **24/7** | Busca todo el día (recomendado) |
| **Respeta horarios** | Solo busca en los horarios que definas |

### 4️⃣ Filtra por Meses (Opcional)

- **Cualquier mes** - Acepta la primera cita disponible
- **Meses específicos** - Solo busca en los meses que selecciones

### 5️⃣ Guarda y Activa

Haz clic en **💾 GUARDAR** y la extensión comenzará a buscar automáticamente.

## ❓ Preguntas Frecuentes

<details>
<summary><b>❓ ¿Es legal usar esta extensión?</b></summary>

✅ **Sí, es completamente legal**. La extensión:
- Solo automatiza el relleno de formularios
- Respeta el límite de 30 intentos/hora del servidor
- NO evade CAPTCHAs automáticamente
- Requiere intervención manual para códigos SMS
- NO realiza acciones maliciosas
</details>

<details>
<summary><b>❓ ¿Puedo ser baneado?</b></summary>

**No.** La extensión respeta todas las limitaciones del servidor. Es como si estuvieras refrescando manualmente, pero automatizado.
</details>

<details>
<summary><b>❓ ¿Funciona en el móvil?</b></summary>

✅ **Sí**. Funciona en edge Browser (Android) y Orion Browser (iPhone).
</details>

<details>
<summary><b>❓ ¿Por qué no automatiza los SMS?</b></summary>

Por **seguridad**. Los códigos SMS son la última barrera de seguridad. La extensión te notifica y espera tu entrada manual.
</details>

<details>
<summary><b>❓ ¿Cuántos intentos por hora hace?</b></summary>

Máximo **30 intentos por hora** (límite del servidor). Al alcanzarlo, espera 5 minutos y reintenta.
</details>

<details>
<summary><b>❓ ¿Puedo usarla en varias pestañas?</b></summary>

**No es recomendable.** Usa una sola pestaña para no saturar el servidor.
</details>

<details>
<summary><b>❓ ¿Guarda mis datos en la nube?</b></summary>

**NO.** Todos tus datos se guardan localmente en `chrome.storage.local`. Nunca salen de tu dispositivo.
</details>

<details>
<summary><b>❓ ¿Qué hago si no encuentra citas?</b></summary>

Las citas se liberan aleatoriamente. Deja la extensión corriendo 24/7 y ten paciencia.
</details>

## 🛠️ Solución de Problemas

| Problema | Solución |
|----------|----------|
| ❌ No se ve el panel | Recarga la página o reinstala la extensión |
| ❌ No guarda configuración | Revisa que el permiso `storage` esté activo |
| ❌ No suena el audio | Haz clic en "🔊 TEST" y permite el sonido |
| ❌ Error en Edge | Usa Chrome (Edge bloquea extensiones no firmadas) |
| ❌ Error "Too Many Requests" | Espera 5 minutos, la extensión reintenta sola |
| ❌ ERR_BLOCKED_BY_CLIENT | Desactiva temporalmente el antivirus |
